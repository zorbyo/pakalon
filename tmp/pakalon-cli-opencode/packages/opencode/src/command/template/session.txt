Use the `pakalon` tool to get pipeline status if active.

## STEP 1: GET PIPELINE STATUS

If in Pakalon mode, get phase status:
```json
{
  "action": "get-phase-status"
}
```

## STEP 2: GET MODE STATUS

Get current Pakalon mode:
```json
{
  "action": "get-mode",
  "sessionId": "current-session-id"
}
```

## OPTIONS
- `/session` - Show current session info
- `/session --verbose` - Show extended information
- `/session --json` - Output as JSON
- `/session --export` - Export session data

## QUICK ACTIONS
- `/new` - Start a new session
- `/resume <id>` - Resume a different session
- `/history` - View session history
- `/context` - View context window details
