You are executing the /figma command for Pakalon.

## Task
Import and analyze designs from Figma for wireframe generation.

## Overview
Figma integration enables:
- Import designs directly from Figma
- Analyze design elements (colors, typography, components)
- Generate wireframes based on Figma designs
- Make minor adjustments if needed
- Proceed to development phases

## Prerequisites
- Figma Personal Access Token (PAT)
- Access to the Figma file

## Setup

### 1. Get Figma Personal Access Token
1. Go to Figma → Settings → Account → Personal Access Tokens
2. Click "Create new personal access token"
3. Copy the token
4. Set in Pakalon:
\`\`\`
/figma setup <your-token>
\`\`\`

### 2. Configure Project
Save token to \`.pakalon/config.json\`:
\`\`\`json
{
  "figma": {
    "token": "your-figma-pat",
    "default_file": "optional-default-file-key"
  }
}
\`\`\`

## Usage

### Commands
\`\`\`
/figma setup <token>           # Set up Figma Personal Access Token
/figma import <file-url>       # Import design from Figma file
/figma analyze <file-url>      # Analyze design without importing
/figma list                    # List imported Figma designs
/figma sync                    # Sync changes from Figma
/figma export                  # Export design assets
\`\`\`

### Import Workflow
1. **Import Design**
   \`\`\`
   /figma import https://www.figma.com/file/abc123/MyDesign
   \`\`\`

2. **Design Analysis**
   Pakalon will extract:
   - Color palette
   - Typography
   - Component library
   - Layout structure
   - Spacing patterns

3. **Review Analysis**

## Figma API Integration

### Endpoints Used
- GET /v1/files/:file_key - Get file data
- GET /v1/files/:file_key/nodes - Get specific nodes
- GET /v1/images/:file_key - Export images
- GET /v1/styles/:file_key - Get styles

## Export Options

### Supported Formats
- SVG (vector graphics)
- PNG (raster images)
- JSON (design tokens)
- CSS (style variables)
- Tailwind (config values)

### Example Export
\`\`\`bash
/figma export --format css
\`\`\`

**Output:**
\`\`\`css
:root {
  --color-primary: #6366f1;
  --color-secondary: #8b5cf6;
  --color-accent: #10b981;
  --font-heading: 'Inter', sans-serif;
  --font-body: 'Inter', sans-serif;
}
\`\`\`

## Security Considerations
- Store Figma token securely in \`.pakalon/config.json\`
- Never commit tokens to version control
- Add \`.pakalon/config.json\` to \`.gitignore\`
- Use environment variables for CI/CD

## Troubleshooting

### Token Invalid
- Regenerate token in Figma settings
- Ensure token has file read permissions

### File Not Found
- Check file URL format
- Verify you have access to the file
- Ensure file is not in trash

### Rate Limiting
- Figma API has rate limits
- Use caching for repeated requests
- Implement exponential backoff
