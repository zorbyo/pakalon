## Usage
- `/history` - Show all sessions for current directory
- `/history <n>` - Show last n sessions
- `/history <session_id>` - Show detailed session info
- `/history --all` - Show sessions from all directories
- `/history --search <query>` - Search sessions by content
- `/resume <session_id>` - Resume a specific session

## Filtering Options
- By date range: `--from 2026-03-01 --to 2026-03-21`
- By model: `--model claude-3.5-sonnet`
- By status: `--status active|completed`
- By directory: `--dir /path/to/project`

## Export Options
- `/history export` - Export history to JSON
- `/history export --format csv` - Export to CSV
- `/history export --format md` - Export to Markdown
