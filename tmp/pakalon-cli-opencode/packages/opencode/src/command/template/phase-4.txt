10. **Nuclei** - Vulnerability scanner

## Docker Commands
```bash
# Run security tools
docker compose -f docker-compose.security.yml up -d

# Run specific tools
docker compose -f docker-compose.security.yml up semgrep
docker compose -f docker-compose.security.yml up zap
docker compose -f docker-compose.security.yml up nikto

# Run full suite
docker compose -f docker-compose.security.yml --profile full up -d
```

## Report Format
Each tool should generate a confidence report:
- Tool name
- Scan duration
- Vulnerabilities found (Critical/High/Medium/Low)
- Confidence score (0-100%)
- Recommendations

## Critical Rules
- Run tools in parallel where possible
- Don't skip any security tools
- Document all findings
- Provide actionable recommendations
