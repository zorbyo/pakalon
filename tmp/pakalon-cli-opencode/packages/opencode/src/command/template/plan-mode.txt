## PLANNING PROCESS

### Step 1: Analyze Request
- Understand the user's request
- Identify the scope and complexity
- Determine required technologies

### Step 2: Create Plan

Generate plan.md in `.pakalon/`:
- Project overview
- Technology stack decisions
- Implementation approach
- Milestones and phases

### Step 3: Create Task List

Generate task.md in `.pakalon/`:
- Break down into specific tasks
- Use checkboxes: `- [ ] Task description`
- Order by dependency
- Group by category (setup, development, testing, deployment)

### Step 4: Create User Stories

Generate user-stories.md in `.pakalon/`:
- US-001, US-002, etc.
- Acceptance criteria
- Test scenarios

### Step 5: Define Context Management

Generate context-management.md:
- Token allocation per task
- Buffer management
- Usage tracking strategy
